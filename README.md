# 🚀 Planly  
### A Student Academic Control Panel  
**Built by Khushi Singh**

---

## 📌 Overview

Planly is a web-based student management system designed to eliminate the cognitive overload caused by fragmented academic information.

Students constantly lose mental bandwidth tracking:

- Attendance  
- Assignment deadlines  
- Timetables  
- Exam dates  

Information is scattered across:

- WhatsApp groups  
- Notice boards  
- PDFs  
- LMS portals  
- Emails  
- Personal notes  

Students don’t miss deadlines because they are careless.  
They miss them because the system is fragmented.

Planly centralises everything into a single, student-first dashboard.

---

## 🎯 Problem Statement

Undergraduate students:

- Waste the first 10 minutes of every day asking, “What do I have today?”
- Struggle to track attendance percentages accurately
- Miss assignments due to scattered communication
- Feel overwhelmed by inconsistent information flow
- Use generic tools (Notion, Google Calendar) not tailored to college workflow

There is no unified academic-specific control panel built for student life.

---

## 💡 Solution

Planly combines:

- Timetable  
- Attendance  
- Assignments  
- Deadlines  
- Events  

Into one structured system.

When the app opens, students immediately see a **Today View**, eliminating friction and decision fatigue.

Planly functions as a lightweight academic operating system.

---

## 🧩 Core Features

### 🏠 Home — Daily Control Dashboard

The Home tab provides an instant overview of the day.

Features:

- Greeting message  
- Dynamic motivational quote  
- Number of classes today  
- Number of deadlines/events today  
- Today & Tomorrow event overview  
- **Must Attend Classes Section**
  - Flags subjects below 75% attendance  
  - Predicts risk if the next class is missed  
- Contact & Feedback integration  

The goal is to reduce cognitive load within seconds.

---

### 📅 Timetable Management

Students can create and manage their weekly timetable.

Visual indicators provide instant clarity:

- Grey → Past classes  
- Green → Ongoing class  
- Blue → Upcoming classes  

This enables quick scanning without mental processing.

---

### 🧠 AI-Powered Planner

Users can paste:

- Plain English messages  
- Forwarded WhatsApp messages  
- Raw, unstructured text  

The system:

- Uses AI parsing (LLaMA via Groq API)  
- Converts text into structured event cards  
- Automatically classifies items into:
  - Today  
  - Tomorrow  
  - Upcoming  
  - Past  
- Categorises entries:
  - Academic  
  - Exams  
  - Clubs  
  - Events  

This removes manual entry friction and converts chaos into structured planning.

---

### 📊 Attendance Tracker

Subject-wise attendance monitoring with:

- Dedicated subject cards  
- Marking attended/missed classes  
- Automatic subject creation if added in Timetable  

Ensures data consistency across the system.
